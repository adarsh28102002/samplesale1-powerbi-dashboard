# 📊 Retail Sales Performance Dashboard (Power BI)

## 🔍 Project Overview

This Power BI dashboard provides a comprehensive analysis of retail sales performance across different regions and product categories. It helps in tracking key business metrics and identifying trends to support data-driven decision-making.

---

## 🎯 Key KPIs

* **Total Sales:** 6.33M
* **Sales MTD (Month-to-Date):** 177.60K
* **MoM Growth:** 2.89%

---

## 📈 Dashboard Insights

* Sales performance is highest in the **West region** compared to other regions.
* **Electronics category** contributes the most to total sales.
* Monthly sales show a **declining trend** over time.
* Average sales vary significantly across months, indicating seasonal demand patterns.

---

## 📊 Features

* Region-wise and category-wise sales analysis
* Monthly sales trend visualization
* KPI cards for quick performance tracking
* Interactive category filter (Electronic, Furniture, Stationery)

---

## 🗂️ Dataset

* Retail sales dataset (sample/business data)
* Includes fields like: Region, Category, Sales, Date
